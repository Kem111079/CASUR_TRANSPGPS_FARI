# Uso en campo · CASUR Transportes GPS V6

## Conductor

1. Abrir la app desde el enlace HTTPS.
2. Tocar **Datos del viaje**.
3. Ingresar conductor, placa/equipo, tipo de viaje, origen y destino.
4. Tocar **Activar GPS**.
5. Tocar **Iniciar recorrido**.
6. Al terminar, tocar **Finalizar recorrido**.
7. Usar **Enviar / Exportar** para descargar PDF, Excel o compartir resumen.

## Supervisor

1. Cambiar a **Modo Supervisor**.
2. Revisar historial de recorridos.
3. Abrir rutas guardadas.
4. Descargar PDF/Excel por recorrido.
5. Exportar consolidado.

Esta versión no incluye simulador. Es la app operativa para recorridos reales.
