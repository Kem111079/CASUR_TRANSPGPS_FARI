# V5.8 · Roles claros con captura y envío

## Problema corregido

En versiones previas el Modo Conductor quedaba demasiado compacto y podía mostrar solo KPIs, ubicación e iniciar recorrido, sin dejar evidente dónde ingresar datos o cómo enviar la información.

## Solución

Se agregan acciones rápidas visibles en modo conductor:

- Datos del viaje
- Enviar / Exportar

Estas acciones funcionan incluso con la bitácora compacta.

## Separación de roles

### Conductor
Captura y envía el recorrido:

- Datos del viaje
- Activar GPS
- Iniciar / Finalizar
- Marcar lugar
- PDF / Excel / WhatsApp

### Supervisor
Revisa y controla:

- Historial
- Consolidado
- Opciones avanzadas
- Rutas guardadas
- Estado de sincronización futura

Esta separación sigue siendo interfaz local de PWA, no autenticación real. La seguridad formal vendrá con app administrador/base central.
